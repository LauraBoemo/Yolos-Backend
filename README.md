### Necessário...

* [Node.js](https://nodejs.org/en/download/) - Linguagem Backend
* [MongoDB](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-windows/) - Banco de Dados
* [Studio3t](https://studio3t.com/download/) - Para utilizar MongoDB

### Hora do processo...

1. Clonando o repositório:
   ```bash
   git clone https://github.com/LauraBoemo/Yolos-Backend
   ```
   
2. Instalando as dependências:
   ```bash
   npm install
   # ou
   yarn
  ```

3. Criando pasta env:
   ```bash
   cp .env.local .env
   ```

4. Iniciando o Servidor:
   ```bash
   yarn start
   # ou
   npm start
   ```

5. Acredito que vá alocar nessa porta, só ir:
   ```bash
   http://127.0.0.1:3001/
   ```

6. Seeding - Inserindo dados fictícios dentro do banco com esse comando:

  ```bash
  yarn seed
  ```
